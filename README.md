# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта
Проект посвящён исследовательскому анализу данных о поведении пользователей сервиса Яндекс Афиша. Цель – выявить факторы, влияющие на возврат клиентов, и дать рекомендации по повышению удержания.

## Данные
- Исходные данные: заказы, мероприятия, города, регионы за период июнь–октябрь 2024 года.
- После предобработки: 287 714 заказов, 21 376 пользователей.

## Основные результаты
- **Лояльность:** 49,1% пользователей совершили 2+ заказа, 18,5% – 5+ заказов.
- **Ключевые факторы возврата:** интервал между заказами (чем короче, тем выше лояльность), среднее количество билетов (оптимум 2–3 билета), тип первого мероприятия (спорт, театр, концерты), билетный оператор (определённые операторы дают более высокую долю возвратов).
- **Рекомендации:** персонализированные предложения для активных сегментов, стимулирование повторных покупок в первые недели, ретаргетинг на пользователей с 1–2 билетами.

## Технологии
- Python (pandas, numpy, matplotlib, seaborn, sqlalchemy, phik)
- Jupyter Notebook
- PostgreSQL
- Git

## Запуск
1. Клонировать репозиторий.
2. Установить зависимости: `pip install -r requirements.txt`.
3. Создать файл `.env` с параметрами подключения к БД.
4. Запустить Jupyter Notebook и выполнить ячейки по порядку.
